### AI-Powered Attack Compromises 600+ FortiGate Firewalls

A Russian-speaking, financially motivated threat actor leveraged commercial AI services to compromise over **600 FortiGate devices across 55 countries** between January 11 and February 18, 2026. This represents a watershed moment in AI-enabled cybercrime.

**Key Details:**
- **AI Tools Used**: The attackers used DeepSeek and Anthropic Claude to generate structured attack plans
- **Attack Method**: Exploited exposed management ports and weak credentials with single-factor authentication (no vulnerability exploitation)
- **Automation**: Claude Code was configured to autonomously execute offensive tools including Impacket scripts, Metasploit modules, and hashcat
- **Geographic Reach**: Concentrated in South Asia, Latin America, Caribbean, West Africa, Northern Europe, and Southeast Asia
- **Significance**: Shows how AI lowers the barrier to entry, enabling even less sophisticated actors to conduct attacks at scale

### Microsoft Copilot and Grok Can Be Abused as Malware C2 Proxies

Check Point Research discovered that AI assistants with web browsing capabilities can be weaponized as covert command-and-control (C2) channels for malware.

**How It Works:**
- Leverages Copilot and Grok's web-browsing/URL-fetch capabilities to retrieve attacker-controlled URLs
- Creates a bidirectional communication channel for malware commands and data exfiltration
- **No authentication required** - works without API keys or registered accounts, making traditional countermeasures ineffective
- AI service domains are treated as routine corporate traffic, evading most conventional detection

**Vendor Response:**
- Microsoft confirmed the research and implemented changes to address the behavior in Copilot's web fetch flow
- As of publication, Grok's URL fetch capabilities remain accessible without authentication

### Ladybird Browser Adopts Rust with AI Assistance

The independent Ladybird web browser project announced on February 23, 2026 that it is adopting Rust with significant help from LLM-based coding assistants.

**Key Achievements:**
- Successfully ported **LibJS** (Ladybird's JavaScript engine) from C++ to Rust in just **2 weeks**
- Resulted in approximately 25,000 lines of Rust code
- Work that would have taken **multiple months** to do manually
- Zero regressions and no performance degradation
- Every AST produced by the Rust parser is identical to the C++ one

**Technical Approach:**
- Andreas Kling used Claude Code and Codex for translation
- Process was human-directed, not autonomous - developer decided what to port, in what order, and what the Rust code should look like through hundreds of small prompts
- Target: LibJS's lexer, parser, AST, and bytecode generator (relatively self-contained with extensive test262 coverage)

**Why Rust:**
- Team searched for memory-safe replacement for C++
- Previously explored Swift but interop and platform support were limited
- Originally rejected Rust in 2024 because "it's not great at C++ style OOP"
- Made the "pragmatic choice" after another year

### Firefox 148 Launches with AI Kill Switch Feature

Mozilla released Firefox 148 on February 24, 2026, delivering on its promise to give users complete control over AI features.

**Key Feature: AI Controls**
- New "AI Controls" section in Settings allows users to disable all AI features
- Access via: Settings > AI Controls > "Block AI Enhancements" toggle
- **Mozilla's Promise**: Future updates will NOT re-enable AI features once disabled

**What Can Be Disabled:**
- ChatGPT and other chatbots in sidebar
- AI link previews
- Smart tab group suggestions
- Translations
- Image alt text in PDF viewer
- Key points in link previews
- AI chatbot providers in sidebar

**Additional Changes:**
- Users can now opt out of Firefox's telemetry and data collection while continuing to receive "remote improvements" between releases
- Drag-and-drop fixes and other improvements
